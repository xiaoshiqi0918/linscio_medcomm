# 门户网站页面详细规划

本文档定义 portal-system 门户前端的视觉规范、路由、首页区块与用户中心页面结构，便于设计与开发对齐。

---

## 10.1 视觉设计规范

**设计方向：** 精准学术科技感。白底灰调为基底，从 Logo 提取深蓝/青绿为核心色，营造「可信赖的医疗 AI 基础设施」气质。完全区别于医院视觉风格。

**色彩体系（从 Logo 提取）：**

| 名称 | 色值 | 用途 |
|------|------|------|
| 主蓝 | `#1A3FCC` | 主按钮 / 链接 / 标题强调 |
| 青绿 | `#00C4AA` | 副按钮 / 成功状态 / 标签 / 代码块左侧竖线 |
| 微紫 | `#7B5FCC` | AI 感装饰色 |
| 渐变主色 | `135deg, #1A3FCC 0% → #00C4AA 100%` | Logo、主视觉装饰 |

| 名称 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#FFFFFF` | 页面底色 |
| 次背景 | `#F6F8FC` | 区块分隔 |
| 卡片背景 | `#EEF2FA` | 卡片、步骤块 |
| 主文字 | `#111827` | 正文标题 |
| 辅文字 | `#4B5563` | 正文次要 |
| 弱文字 | `#9CA3AF` | 提示、说明 |
| 边框线 | `#E2E8F2` | 分割线、输入框边框 |

| 名称 | 色值 |
|------|------|
| 成功色 | `#10B981` |
| 警告色 | `#F59E0B` |
| 危险色 | `#EF4444` |

**字体方案：**

| 用途 | 字体 |
|------|------|
| 英文标题 | DM Serif Display（衬线，学术气质） |
| 中文标题 | Noto Serif SC（与英文标题协调） |
| 正文 | DM Sans + Noto Sans SC |
| 代码 | JetBrains Mono |

**组件规范：**

| 项目 | 规范 |
|------|------|
| 圆角 | 卡片 12px / 按钮 8px / 标签 999px |
| 阴影 | 默认 `0 2px 8px rgba(0,0,0,0.06)`；悬停 `0 8px 32px rgba(26,63,204,0.10)` |
| 导航栏 | 高度 64px，固定顶部；滚动后 `backdrop-filter: blur(16px)` |
| 代码块 | 深色背景 `#0F172A`，左侧青绿竖线，JetBrains Mono |

**装饰元素：**

- Hero 区：极淡蓝灰网格线（`#E8EDF8`，1px，间距 40px）
- 页面角落：Logo 同色系径向渐变光晕（青绿透明度约 8%）

实现对应：`portal/src/assets/global.css` 中 `:root` 变量；`index.html` 中引入上述字体。

## 10.2 路由规划

| 路由 | 页面 | 访问要求 |
|------|------|----------|
| `/` | 首页 | 公开 |
| `/features` | 功能详情页 | 公开 |
| `/auth` | 注册/登录 | 未登录可访问 |
| `/dashboard` | 用户中心 | 需登录 |
| `/docs` | 文档页 | 公开 |
| `/about` | 关于我们 | 公开 |

导航栏链接：产品概览 → `/`；功能详情 → `/features`；安装文档 → `/docs`；关于我们 → `/about`。右侧：未登录为「登录」「免费注册」；已登录为用户名/头像 + 用户中心、退出。

## 10.3 首页区块规划（从上到下）

### ① 导航栏（固定顶部）

- **左：** 圆形 Logo 图标 +「linscio AI」字样  
- **中：** 产品概览 | 功能详情 | 安装文档 | 关于我们  
- **右：** 未登录：[登录] [免费注册]；已登录：用户名 + 头像下拉（用户中心 / 退出）

### ② Hero 区

- **背景：** 白底 + 极淡蓝灰网格线 + 右上角青绿光晕  
- **左侧（约 55%）：**  
  - 大标题（DM Serif Display 52px）：**AI 驱动的医学知识引擎**  
  - 中文副标（Noto Serif SC 20px，蓝色）：专为科研而生，本地部署，数据永不离境  
  - 描述（16px 灰色）：面向医学科研人员的本地化 AI 智能体集群，集文献检索、论文写作、数据分析、科普创作于一体  
  - 按钮组：[立即免费注册]（主蓝）、[查看功能详情 →]（描边）  
- **右侧（约 45%）：** 使用 **纯 SVG + CSS** 实现的粒子呼吸动画（主蓝 #1A3FCC、青绿 #00C4AA 圆点，错峰 opacity 关键帧），无图片、无 Lottie 依赖，保证任意环境可见。实现见 `portal/src/views/Home.vue` 内 `.hero-lottie` 与 `@keyframes particle-pulse`。  
- **底部信任指标条：** 33 个专业智能体 | 6 大业务模块 | 本地私有部署 | 数据完全自主  

### ③ 痛点共鸣区

- **背景：** 次背景色 `#F6F8FC`  
- **标题：** 科研路上，这些困境你是否熟悉？  
- **布局：** 3 列卡片，左侧蓝色竖线装饰  
- **卡片内容：**  
  - 📚 文献管理失控 — 文献散落各处，手动整理耗时，检索效率极低  
  - ✍️ 写作效率低下 — 从选题到投稿耗时数月，格式规范反复修改  
  - 📊 数据分析门槛高 — SPSS 操作繁琐，统计方法不确定，图表制作费力  

### ④ 六大产品概览

- **标题：** 六大智能模块，覆盖医学科研全场景  
- **布局：** 2×3 卡片网格；每卡：产品图标（蓝/青渐变）、英文名（DM Serif Display）、中文定位（加粗）、核心能力一句话（小字灰色）、[了解详情 →]  
- **六张卡片：**  
  - LinScio Schola — 学术论文智能辅助系统  
  - MedComm Creator — 医学科普内容创作  
  - QCC Assistant — 质量控制智能助手  
  - AI Image Studio — 一站式 AI 图像创作工作台  
  - LinScio Literature — 文献管理与检索  
  - Data Analyzer — 数据分析与统计  

### ⑤ 核心优势区

- **标题：** 为什么选择 LinScio AI？  
- **布局：** 2×2 网格，大图标 + 标题 + 说明  
- **四条：**  
  - 🔒 本地部署，数据不上云  
  - 🤖 33 个专业智能体协同  
  - ⚡ 多阶段工作流引擎  
  - 🔄 智能 LLM 三级降级  

### ⑥ 部署三步骤

- **标题：** 三步完成部署，10 分钟即可开始使用  
- **背景：** 次背景色  
- **三步：** Step 1 注册并激活（访问门户→注册→授权码→Registry 凭证） / Step 2 拉取镜像（docker login / docker pull） / Step 3 启动服务（docker-compose up -d，访问 localhost:8080）  

### ⑦ CTA 结尾区

- **背景：** 渐变 `#1A3FCC → #0095A0`  
- **标题（白）：** 立即开始，让 AI 成为你的科研伙伴  
- **副标（半透明白）：** 注册账号，填写授权码，即可获取专属镜像  
- **按钮：** [立即注册账号]（白底蓝字）  

### ⑧ 页脚

- **左：** Logo + 一句话定位  
- **中：** 快速导航 / 功能模块 / 帮助文档  
- **右：** 联系方式  
- **底：** © 2026 LinScio AI | ICP备XXXXXXXX号
