## 小结

- **与表一致**：激活、作废、延期、模块权限变更、拉取/生成次数超限、机器绑定满、并发超限、授权自然到期（定时清理 + 门户/Registry 立即失效）、手动重置配额（生成次数）。
- **差异说明**  
  - **授权自然到期 → 应用实例**：当前到期后心跳仍返回 200 且 `license_expired=true`，无「7 天宽限后心跳 403」；若产品要求 7 天后拒绝启动，可在心跳中按 `expires_at + 7 天` 判断并返回 403。  
  - **手动重置配额 → 拉取次数**：拉取次数按自然月重置，无单独管理端「重置拉取次数」接口；若需可新增（如将某用户/凭证的 pull_count_this_month 置 0）。
