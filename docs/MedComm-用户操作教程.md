## 1. 安装与首次启动

### 1.1 下载与安装

1. 前往 MedComm 官网 `medcomm.linscio.com.cn`，点击 **下载**
2. 根据操作系统选择对应安装包：
   - **macOS**：`.dmg` 文件，双击后拖入「应用程序」文件夹
   - **Windows**：`.exe` 安装程序，按向导完成安装
3. 首次打开可能提示安全警告，按系统引导允许运行即可

[截图：MedComm 官网下载页面]

### 1.2 首次启动

启动后，软件会依次完成以下初始化：

1. **启动画面（Splash）**：显示加载进度
2. **依赖检查与数据库迁移**：自动完成，无需操作
3. **本地服务启动**：等待内置 AI 服务就绪

> 首次启动可能需要 30–90 秒（需安装依赖），后续启动通常在 10 秒以内。

[截图：Splash 启动画面]

### 1.3 选择本地用户名

首次进入主界面时，会弹出对话框要求输入**本地用户名**。此用户名用于本地数据隔离（不同用户拥有独立的文档和设置），与门户账号无关。

[截图：选择/创建本地用户名弹窗]

## 2. 激活授权

MedComm 客户端必须完成授权激活后才能使用。有两种激活方式：

### 方式一：门户账号登录（推荐）

1. 在激活引导页，点击 **使用门户账号登录**
2. 输入在门户网站 `portal.linscio.com.cn` 注册的**邮箱**和**密码**
3. 点击 **登录**
4. 登录成功后自动跳转到主界面

[截图：激活引导页 - 门户账号登录表单]

### 方式二：官网激活（Deep Link）

1. 在激活引导页，点击 **前往官网激活**
2. 浏览器自动打开门户网站的授权激活页面
3. 在门户网站完成登录和激活操作
4. 门户会通过 `linscio://` 协议链接自动唤起客户端并完成绑定

[截图：激活引导页 - 两种激活选项]

> **提示**：如果还没有门户账号，点击激活页底部的 **前往门户网站** 链接注册。

## 4. 科普写作

科普写作是 MedComm 的核心功能，提供从选题到发布的全流程 AI 辅助写作体验。

### 4.1 新建文章

1. 点击侧栏 **科普写作**，或点击左侧文章列表上方的 **+ 新建** 按钮
2. 进入新建文章表单页

[截图：新建文章页面全貌]

#### 选择内容形式

MedComm 支持多种科普内容形式：

| 形式 | 说明 |
|------|------|
| 图文文章 | 适用于公众号、头条等平台的标准科普文 |
| 辟谣文 | 针对常见健康误区的科普辟谣 |
| 口播脚本 | 短视频口播稿 |
| 条漫脚本 | 条漫形式科普（含分格提示） |
| 绘本脚本 | 儿童健康绘本 |
| 患者手册 | 疾病管理手册（含警示框、提示框） |
| 自测题 | 互动式健康自测 |

可通过 **平台优先** / **形式优先** 开关切换选择逻辑：
- **形式优先**：先选内容形式，再选目标平台
- **平台优先**：先选平台（如微信公众号），系统推荐适合的形式

[截图：形式选择区域（FormatPicker）]

#### 填写文章信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 主题 | 是 | 文章的核心主题，如「糖尿病患者的饮食管理」 |
| 目标受众 | 否 | 目标读者群体 |
| 阅读难度 | 否 | 面向人群的医学背景水平 |
| 选择专科 | 是 | 对应医学专科（定制版显示已购学科包标识） |
| 模板 | 否 | 选择预设结构模板 |

3. 填写完成后点击 **创建**

[截图：填写主题和受众的表单区域]

[截图：专科选择下拉列表]

### 4.3 AI 智能生成

1. 在编辑器中选择要生成内容的**章节**
2. 点击工具栏的 **AI 生成** 按钮
3. AI 会根据主题、受众、形式和参考文献**流式生成**内容
4. 生成过程中内容实时显示在编辑器中
5. 生成完成后可手动编辑调整

[截图：AI 生成过程中的流式输出效果]

> **注意**：AI 生成需要已配置有效的 LLM API Key（见 [11.2 API Key 配置](#112-api-key-配置)）。

### 4.4 参考文献引用

在文章编辑页面，可展开底部的 **参考文献绑定** 区域：

[截图：参考文献绑定区域（折叠状态）]

#### 添加本地文献

1. 点击 **添加文献**
2. 弹出本地文献库搜索对话框，支持多选
3. 选中后点击确认，文献绑定到当前文章

[截图：从本地文献库选择文献的弹窗]

#### 检索外部文献

1. 点击 **检索支撑文献**
2. 弹出检索对话框，支持多个数据源：
   - **PubMed**：生物医学文献
   - **CrossRef**：跨学科文献
   - **Semantic Scholar**：AI 驱动的学术搜索
3. 设置检索参数：关键词、年份范围、语言
4. 查看搜索结果，选择需要的文献
5. 点击导入，自动存入本地文献库并绑定到文章

[截图：外部文献检索对话框]

#### 引用范围

可切换引用范围：
- **当前章节**：仅关联到当前编辑的章节
- **整篇文章**：关联到整篇文章

---

### 4.5 导出文章

1. 点击工具栏的 **导出** 下拉按钮
2. 选择导出格式：

| 格式 | 适用场景 |
|------|---------|
| HTML | 网页发布 |
| DOCX | Word 编辑 |
| PDF | 打印和分享 |
| TXT | 纯文本 |

3. 导出前系统会自动进行**导出检查**：
   - 检测 `[DATA:]` 占位符（未填内容）
   - 检测绝对化表述（如「一定」「绝对」）
   - 检测孤儿引用（引用了但未绑定文献）
4. 如有问题，可点击定位修改，或选择 **仍要导出**

[截图：导出前检查对话框]

## 6. 文献支撑库

文献库是 MedComm 的学术文献管理模块，支持本地存储、外部检索和 PDF 阅读。

[截图：文献支撑库主界面全貌]

### 界面结构

| 区域 | 说明 |
|------|------|
| 左侧栏 | 全部文献 / 未读 / 回收站 + 集合树 + 标签列表 |
| 主区域 | 文献列表（表格形式） + 搜索、排序、筛选 |

---

### 6.1 添加文献

支持多种方式添加文献：

#### 通过 DOI / PMID 添加

1. 点击 **添加文献**
2. 切换到 **DOI/PMID** Tab
3. 输入 DOI 或 PMID，系统自动拉取元数据
4. 确认信息后点击添加

[截图：通过 DOI 添加文献的对话框]

#### 手动录入

1. 点击 **添加文献**
2. 切换到 **手动录入** Tab
3. 填写标题、作者、期刊、年份等信息
4. 点击添加

#### 导入文件

1. 点击 **导入**
2. 支持格式：RIS、BibTeX、PDF
3. 系统自动解析并入库

[截图：导入文献按钮与支持格式]

#### 网页采集

1. 点击 **网页采集**
2. 输入文献页面 URL、标题等信息
3. 提交后自动存入文献库

[截图：网页采集对话框]

---

### 6.2 外部检索

1. 在文献库页面点击 **外部检索**
2. 选择数据源（PubMed / CrossRef / Semantic Scholar）
3. 输入检索关键词，设置年份、语言等筛选条件
4. 检索以**流式**方式返回结果
5. 选中需要的文献，点击保存入库

[截图：外部检索界面 - 检索中]

[截图：外部检索界面 - 结果列表]

> **提示**：使用 PubMed 和 Semantic Scholar 检索需要配置对应的 API Key（见 [11.2 API Key 配置](#112-api-key-配置)），可在标题栏快速配置。

### 11.1 账户与门户授权

#### 本地用户

本地用户用于数据隔离，与门户账号独立。

| 操作 | 说明 |
|------|------|
| 登录 | 未登录时输入用户名登录 |
| 切换用户 | 切换到其他本地用户 |
| 退出登录 | 退出当前本地用户 |

[截图：本地用户区域 - 已登录状态]

#### 门户授权

显示当前授权绑定状态：

| 状态 | 显示内容 |
|------|---------|
| 已绑定 | 门户邮箱、授权到期日 |
| 未绑定 | 邮箱和密码输入框 + **登录绑定** 按钮 |

- **登录绑定**：输入门户邮箱和密码，点击登录即可绑定
- **解除授权绑定**：解除当前设备与门户账号的绑定
- **前往门户注册**：打开浏览器跳转门户网站

[截图：门户授权区域 - 未绑定状态（含邮箱密码输入框）]

[截图：门户授权区域 - 已绑定状态]

#### 版本信息

- 显示当前为**基础版**或**定制版**

### 11.2 API Key 配置

MedComm 需要配置 AI 服务的 API Key 才能使用智能写作和图像生成功能。

[截图：API Key 配置区域]

#### 支持的服务商

| 类型 | 服务商 | 说明 |
|------|--------|------|
| 文本生成 | OpenAI、硅基流动、Deepseek、OpenRouter 等 | AI 写作核心能力 |
| 图像生成 | 硅基流动等 | 配图生成 |
| 学术检索 | NCBI (PubMed)、Semantic Scholar | 文献检索增强 |

#### 配置步骤

1. 在对应服务商栏位输入 API Key
2. 点击 **测试** 验证 Key 是否有效（OpenAI 兼容接口支持）
3. 点击 **保存** 将 Key 安全存储到系统 Keychain
4. 保存后会自动重载后端服务

[截图：输入 API Key 并测试]

> **安全说明**：API Key 存储在操作系统的安全 Keychain 中（macOS Keychain / Windows 凭据管理器），不以明文保存。

标题栏右侧也显示 NCBI Key 和 S2 Key 的状态，点击可快速配置。

### 11.4 数据备份与恢复

[截图：数据备份与恢复区域]

#### 完整备份

1. 点击 **完整备份**
2. 系统自动将所有数据（文章、文献、设置等）打包为 `.linscio-backup` 文件
3. 选择保存位置

#### 恢复备份

1. 点击 **选择备份文件恢复**
2. 选择 `.linscio-backup` 文件
3. 选择恢复策略：
   - **覆盖**：用备份数据替换当前数据
   - **仅补充缺失**：只恢复当前不存在的数据

## 13. 快捷键与常见问题

### 常见问题

**Q: 提示「No handler registered for 'portal-login'」**
A: Electron 主进程未加载最新代码，请完全退出 MedComm 后重新启动。

**Q: 登录绑定提示「Not Found」**
A: 门户 API 服务需要更新到最新版本，请联系管理员。

**Q: 启动后提示「前端构建产物缺失」**
A: 安装包损坏，请重新下载安装。

**Q: 文献检索无结果**
A: 请检查是否已配置对应的 API Key（NCBI Key 或 S2 Key）。

**Q: AI 写作提示错误**
A: 请在设置中检查 LLM API Key 是否有效，点击「测试」验证。

**Q: 如何解除设备绑定？**
A: 进入 设置 → 门户授权 → 点击 **解除授权绑定**。
