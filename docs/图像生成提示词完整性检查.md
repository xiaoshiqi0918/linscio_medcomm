## 9.1 MedPromptBuilder 核心逻辑

### 规范要求

- **STYLE_SYSTEM_PROMPTS**：7 种风格（medical_illustration, flat_design, realistic, cartoon, data_viz, comic, picture_book）
- **QUALITY_SUFFIX**、**SAFETY_NEGATIVE**
- **build()** 流程：
  1. `_translate_if_chinese`：中文→英文
  2. `_standardize_medical_terms`：医学术语标准化
  3. `_get_audience_modifier`：受众风格倾向
  4. 组合：base_desc + audience + style + QUALITY_SUFFIX
  5. 负向提示词：text overlay, watermark, SAFETY_NEGATIVE
- 返回 `(positive_prompt, negative_prompt)`

### 当前实现

- **无 MedPromptBuilder** 类
- **build_med_prompt_node**（`image_nodes.py`）：
  - 从 DB 取 ImagePromptTemplate，或
  - `style_hint（中文）+ desc`，无翻译、无术语标准化

## 9.2 各图像类型的专属提示词模板

### 规范要求

| image_type | en_template 占位符 | negative |
|------------|-------------------|----------|
| anatomy | organ_or_structure, specific_focus | 解剖失真 |
| pathology | disease, mechanism | 血腥病理 |
| flowchart | process_name | 杂乱、流程不清 |
| infographic | topic, data_points | 排版混乱 |
| comparison | healthy_state, disease_state | 血腥对比 |
| symptom | condition | 血腥症状 |
| prevention | healthy_behavior | 负面画面 |
| comic_panel | scene_desc, emotion | 西式、暴力 |
| picture_book_page | illustration_desc | 恐怖、成人 |
| storyboard_frame | scene_desc, camera_note | 写实、复杂 |

### 当前实现

- **无 IMAGE_TYPE_TEMPLATES**
- image_types.py 有 IMAGE_TYPES 列表，但无模板
- build_med_prompt 未按 image_type 选用模板

## 9.3 配图建议生成（suggest_images_node）

### 规范要求

- **SUGGEST_IMAGES_PROMPT**：LLM 分析 verified_content
- 输出 JSON：anchor_text, image_type, style, description, en_description, reason, priority
- 规则：何时推荐/不推荐、最多 3 处

### 当前实现

- **suggest_images_node**：按 `\n\n` 分段，段落 >80 字则加入建议，无 LLM
- 输出结构：index, summary, anchor_text（无 image_type, description, en_description 等）
