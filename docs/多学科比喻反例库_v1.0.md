## 一、使用说明

### 1.1 比喻准确性判断方法（v2.0 规则五）

```
判断一个医学比喻是否合格，做三步还原检验：

Step 1：把比喻还原为字面含义
Step 2：检查字面含义是否与真实医学机制方向一致
Step 3：检查比喻是否会让读者产生错误的治疗期望

三步全部通过 → 比喻合格
任一步骤出现问题 → 比喻需要修改或删除
```

### 1.2 本文档的组织结构

每个科室提供：
- **❌ 典型错误比喻**：LLM 容易生成、看起来合理但实际有误的比喻
- **错误原因**：从医学机制角度解释为何有误
- **✅ 修正比喻**：可以替换使用的准确比喻（或说明为何不宜使用比喻）
