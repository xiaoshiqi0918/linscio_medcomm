## 二、组装结构对照

### 规范要求（1.2）

```python
system_prompt = Layer 0 + Layer 1
user_prompt = RAG + 示例 + 词典 + 任务提示词
```

### 当前实现

| 项 | 规范 | 实现 | 符合 |
|----|------|------|------|
| system_prompt | Layer 0 + Layer 1 | `base.py`: `MEDCOMM_SYSTEM = join([MEDCOMM_SYSTEM_PROMPT, MEDCOMM_ANTI_HALLUCINATION])` | ✅ |
| user_prompt 顺序 | RAG → 示例 → 词典 → 任务 | `prompt_builder`: `[rag_section, example_section, term_section, base_prompt]` | ✅ |
| 方法名 | `build_full_prompt()` | 逻辑内置于 `BaseAgent.run()`，无独立方法 | ⚠️ 语义等价 |

**说明**：规范中的 `build_full_prompt()` 与 `enhancement_orchestrator.build(state)` 在当前实现中分别对应 `run()` 内联逻辑与 `build_enhanced_prompt()` 函数，功能一致。
